# Recipe Manager Prompts

These 4 prompts implement the Recipe Manager business logic.

---

## Prerequisites

Execute starter-kit prompts (001-003) first.

---

## Prompts

**001 - Backend CRUD**
- Recipe model (name, ingredients, instructions, cuisine, difficulty, times)
- Service, controller, routes
- MongoDB with validation

**002 - AI Integration**
- OpenAI service for cuisine and difficulty classification
- Analyzes ingredients and techniques
- AI-powered suggestions

**003 - Frontend React**
- RecipeList, RecipeForm components
- Orange/green food-themed UI (Tailwind)
- AI classification interface

**004 - Testing Implementation**
- Jest (backend) + Vitest (frontend)
- Comprehensive test suites for Recipe logic

---

## Usage

After starter kit, execute in order (001 → 002 → 003 → 004).

Result: Full Recipe Manager application.

---
