# Starter Kit Prompts

These 3 prompts create the reusable foundation for any MERN + AI project.

---

## Prompts

**001 - Project Setup**
- Monorepo structure (backend/frontend)
- TypeScript strict configuration
- ESLint and tooling

**002 - AI Ecosystem Setup**
- llms.txt, AGENTS.md, .cursorrules
- .skills/ folder

**003 - Global Skills Installation**
- Superpowers collection (14 skills)
- wshobson/agents collection (146+ skills)

---

## Usage

Execute in order (001 → 002 → 003) to create starter kit.

Result: Infrastructure ready for your business logic.
